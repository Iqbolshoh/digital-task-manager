# ✨ TaskNova - Digital Task Manager

TaskNova is a modern and lightweight task management web application built with **HTML, CSS, and JavaScript**. It helps users organize daily tasks, increase productivity, and stay focused through a clean and user-friendly interface.

---

## 🚀 Features

* ➕ Add new tasks instantly
* 📋 View all tasks in a clean workspace
* 🎨 Modern and responsive design
* ⚡ Fast and lightweight performance
* 🖥️ Pure HTML, CSS, and JavaScript
* 📱 Mobile-friendly layout
* 🌙 Clean UI focused on productivity

---

## 📸 Preview

TaskNova provides a beautiful landing page with:

* Modern navigation header
* Hero section with call-to-action
* Interactive task management area
* Professional footer
* Responsive design for all devices

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Use

1. Open the application in your browser.
2. Enter a task in the input field.
3. Click the **Add** button.
4. Manage your daily tasks efficiently.
